# Changelog

## [2026.06.14] — Initial Public Release

### 新增
- 全面支持 SM8450/SM8550/SM8650/SM8350 平台
- WALT 窗口辅助负载跟踪
- MIGT 智能任务迁移引导
- Metis 调度器反馈环路
- Phase 多级优先级调度
- IPC 感知调度器协同
- Binder 优先级继承（防优先级反转）
- SukiSU root 方案 + SUSFS 文件系统伪装
- AnyKernel3 打包格式

### 已知问题
- KONA (SM8250/4.19) 平台仍在适配中
- 部分厂商定制 ROM 可能需要额外 dtb 配置
